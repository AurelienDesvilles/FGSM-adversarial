# FGSM-adversarial
As part of the UE RCP209 of CNAM Paris, this project consists testing resistance to FGSM adversarial attacks for tree image recognition models pre-trained on imagnet:
-VGG19 
-ResNet50
-InceptionV3
The image dataset can be found at: https://www.kaggle.com/datasets/deeptrial/miniimagenet/data
